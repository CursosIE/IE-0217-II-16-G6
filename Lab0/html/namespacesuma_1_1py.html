Detailed Description

Este programa realiza la suma de datos introducidos en la consola al ejecutar el mismo.

Obtiene numeros introducidos por el usuario seguidamente al ejecutar el programa suma.c . El usuario debe recordar compilar el programa antes de ejecutarlo. Este programa cuenta con manejo de excepcione en caso de ingresar un dato no valido para la suma.
